CARGOMATE

Smart Robotic Inventory System with Cloud Integration — An IoT-enabled warehouse automation platform with digital twin technology, ML-based quality control, and autonomous navigation.

IoT Cloud Robotics Digital Twin Machine Learning ESP32 Firebase

DOMAIN

Research & Technical Domain

A comprehensive study of four integrated subsystems addressing the key challenges of modern warehouse automation.

Literature Survey

Prior work covers cloud robotics, RBAC, digital twin technology, MQTT protocols, CNN-based barcode detection, RFID navigation, and robotic manipulation.

Research Gap

  • Fragmented system architectures
  • Passive digital twin systems
  • No intelligent barcode quality checking
  • Weak tactile feedback in low-cost grippers
  • Insufficient multi-sensor navigation

Research Problem

How can a secure, scalable, low-latency cloud-native framework manage autonomous warehouse operations using digital twin, ML quality control, robotic manipulation, and hybrid navigation?

Research Objectives

  • Cloud warehouse management with RBAC
  • Barcode scanning with ML quality assessment
  • Adaptive robotic manipulation
  • Hybrid autonomous navigation
  • Experimental validation

Methodology

  • ESP32 / ESP32-CAM embedded systems
  • Firebase Realtime Database
  • TinyML barcode classification
  • FSR + TOF sensor fusion
  • IR + RFID + Ultrasonic navigation

Technologies Used

  • Cloud: Firebase, Grafana
  • Frontend: HTML, CSS, JavaScript
  • ML: MobileNet CNN
  • Sensors: FSR, TOF, IR, RFID, Ultrasonic
